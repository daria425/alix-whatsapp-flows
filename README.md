# WhatsApp Signposting Service

A WhatsApp-based application I am working on as part of my internship at <a href="https://www.projectalix.com">Alix</a>
</br>

- The tool combines Twilio Programmable Messaging with a REST API in order to send automated whatsapp messages to users and signpost them to information on support organizations and charities specified for their query, from a database of over 250 verified organizations.
- Additionally, the tool uses the Gemini API to create text descriptions of each option, as well as translate these into the users preferred language.
- More information is available here: https://www.projectalix.com/cornwall
